D3.js Bar Chart

A freeCodeCamp Project

About this project

This app makes an AJAX request for JSON data from the Federal Reserve Economic Data. Once the data is received, the data is bound to an <svg> element and visualized using the D3.js library. You can mouseover the bars to find detailed information regarding that particular data point.

User Story Requirements

  1. I can see US Gross Domestic Product by quarter, over time.
  2. I can mouse over a bar and see a tooltip with the GDP amount and exact year and month that bar represents.
